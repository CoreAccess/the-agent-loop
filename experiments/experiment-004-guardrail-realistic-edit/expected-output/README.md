# Expected Output Shape

This folder intentionally does not prescribe exact final code.

The completed run should include:

- updated `project/src/notekeeper.py`;
- updated `project/tests/test_notekeeper.py`;
- updated `project/docs/usage.md`;
- updated `project/STATUS.md`;
- appended `project/memory/log.md`;
- `output/action-trace.md`;
- `output/handoff.md`.

The reviewer should evaluate the run using `evaluation/rubric.md`.
