# Reflect Checklist

Before finishing meaningful work:

1. Compare completed work against `.agent-loop/GOAL.md`.
2. Record verification evidence or explain skipped verification.
3. Update `.agent-loop/STATUS.md` with handoff state.
4. Append notable events to `.agent-loop/memory/log.md`.
5. Identify cleanup performed and cleanup still unresolved.
6. Draft durable memory candidates in `.agent-loop/memory/proposed/` only when useful.
7. Ask before promoting, deleting, superseding, or rewriting durable memory.
