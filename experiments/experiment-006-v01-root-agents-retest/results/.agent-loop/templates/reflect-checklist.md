# Reflect Checklist

Before finishing meaningful work:

1. Compare completed work against `.agent-loop/GOAL.md`.
2. Record verification evidence or explain skipped verification.
3. Update `.agent-loop/STATUS.md` with handoff state.
4. For onboarding work, confirm root `AGENTS.md` exists or was carefully updated to load `.agent-loop/AGENTS.md`.
5. Append notable events to `.agent-loop/memory/log.md`.
6. Identify cleanup performed and cleanup still unresolved.
7. Draft durable memory candidates in `.agent-loop/memory/proposed/` only when useful.
8. Ask before promoting, deleting, superseding, or rewriting durable memory.
