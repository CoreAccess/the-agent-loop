# Private Memory Notes

Use this folder only when the active Goal Packet and a human reviewer explicitly approve storing access-sensitive project notes.

## Rules

- Do not store credentials, tokens, API keys, secrets, or unredacted PII.
- Prefer redacted source references over copying sensitive content.
- Do not load this folder by default.
- Any private note still needs a source stamp, purpose, and review status.

If a candidate contains sensitive data and the storage location is unclear, reject it and escalate for review.
